# Homebrew Tap

This repository is the shared Homebrew tap for tasuku43 CLI tools.

## Formulae

- `kra`
- `gion`
- `cc-bash-guard`
- `vivi`

## Install

```sh
brew tap tasuku43/tap
brew install kra
brew install gion
brew install cc-bash-guard
brew install vivi
```

You can also install without tapping first:

```sh
brew install tasuku43/tap/kra
brew install tasuku43/tap/gion
brew install tasuku43/tap/cc-bash-guard
brew install tasuku43/tap/vivi
```
